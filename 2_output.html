Podemos mostrar dados de diferentes formas em JS:

  1. .escrevendo dentro de um elemento html usando "innerHTML"
  2. .escrevendo no documento html atraves da funcao "document.write()"
  3. .escrever dentro de uma caixa de alerta usando o metodo "window.alert()"
  4. .escrevendo dentro do console do browser usando "console.log()"
