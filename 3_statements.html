.Um programa de computador nada mais é do que uma sequencia de comandos/instruções que são executados pelo computador,essas instruções são chamadas de "DECLARAÇÔES" em linguagem de programação.

.Em JS, o browser executa o programa(lista de instruções)

.Em JavaScript,comando são separados por ";" (ponto-virgula)

Exemplos:

let a, b, c; // Declare 3 variables
a = 5; // Assign the value 5 to a
b = 6; // Assign the value 6 to b
c = a + b; // Assign the sum of a and b to c
